# StatBox - 统计学家工具箱

> 一个基于 Wails + Vuetify3 的桌面应用，帮助统计学专业学生高效管理代码、收藏和常用命令。

## 📋 项目概述

StatBox 是为统计学专业数理统计方向学生设计的效率工具，解决代码管理分散、资源收藏混乱、重复输入频繁等痛点问题。

**技术栈：**
- 后端：Go 1.25+ (Wails v2.11)
- 前端：Vue 3 + Vuetify 3 + TypeScript
- 数据存储：JSON + Markdown + 原始代码文件

---

## ✨ 核心功能

### 1. 📁 收藏夹管理

**功能特性：**
- ✅ 分层文件夹结构管理网址收藏
- ✅ 内嵌浏览器支持多标签页浏览
- ✅ 智能打开策略（Kaggle等自动外部打开）
- ✅ 右键菜单：编辑、外部浏览器打开、删除
- ✅ 新建文件夹、添加收藏功能
- ✅ 排序功能：按名称、添加时间、访问频率排序
- ✅ 搜索过滤：按文件夹名、网址名、URL检索

**数据存储：** `~/.statbox/bookmarks.json`

---

### 2. 💻 代码模板管理

**功能特性：**
- ✅ 支持多种语言：R / Python / Julia
- ✅ 查看模式：语法高亮显示代码
- ✅ 运行功能：应用内终端执行
- ✅ 终端输出：可调整大小，字体优化
- ✅ 外部IDE打开（使用系统默认应用）
- ✅ 一键复制代码

**数据存储：** `~/.statbox/templates/`

---

### 3. ⌨️ 命令库

**功能特性：**
- ✅ 快捷唤醒：默认 `~` 键（可自定义）
- ✅ 智能匹配：实时搜索
- ✅ 回车执行：复制命令到剪贴板
- ✅ 唤醒范围可配置：全局/仅应用内
- ✅ 一键复制命令

**数据存储：** `~/.statbox/commands/`

---

### 4. ⚙️ 系统设置

**常规设置：**
- ✅ 开机自启动
- ✅ 静默启动（开机时最小化到托盘）
- ✅ 最小化到系统托盘

**外观设置：**
- ✅ 5种颜色主题可选（温暖蓝、翡翠绿、薰衣草紫、珊瑚橙、玫瑰红）
- ✅ 编辑器字体大小调整

**快捷键设置：**
- ✅ 命令库快捷键（默认 `~`）
- ✅ 命令库唤醒范围（全局/仅应用内）

## 🎨 界面设计

### 布局结构

```
┌─────────────────────────────────────────┐
│  [≡] StatBox                  [-] [□] [×]│
├────────┬────────────────────┬───────────┤
│ 📁收藏 │                    │ 🔍搜索... │
│ 💻代码 │   主显示面板       │ ├─统计方法│
│ ⌨️命令 │  （网页/代码编辑器/ │ ├─R代码   │
│ ⚙️设置 │   命令详情/设置）   │ └─...     │
│        │                    │           │
│[可折叠]│                    │ [可折叠]  │
└────────┴────────────────────┴───────────┘
```

### 界面特性

- ✅ **可拖动标题栏** - 无边框窗口设计，标题栏可拖动
- ✅ **可折叠侧边栏** - 左右两侧面板都可折叠
- ✅ **多种颜色主题** - 5种精美主题可选
- ✅ **响应式布局** - 列表点击即可显示内容

## 🚀 快速开始

### 环境要求

- Go 1.21+
- Node.js 18+
- Wails CLI v2.8+

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd StatBox/statbox

# 安装前端依赖
cd frontend
npm install

# 开发模式运行
wails dev

# 构建生产版本
wails build
```

### 静默启动

```bash
# 静默启动（最小化到托盘）
statbox.exe --silent
```
