# StatBox - 统计学家工具箱

> 一个基于 Wails + Vuetify3 的桌面应用，帮助统计学专业学生高效管理代码、收藏和常用命令。

## 📋 项目概述

StatBox 是为统计学专业数理统计方向学生设计的效率工具，解决代码管理分散、资源收藏混乱、重复输入频繁等痛点问题。

**技术栈：**
- 后端：Go 1.21+ (Wails v2)
- 前端：Vue 3 + Vuetify 3 + TypeScript
- 数据存储：JSON + Markdown + 原始代码文件

---

## ✨ 核心功能

### 1. 📁 收藏夹管理

**功能特性：**
- ✅ 分层文件夹结构管理网址收藏
- ✅ 内嵌浏览器（WebView）支持多标签页浏览
- ✅ 标签页管理：关闭、刷新、复制URL、保存网页
- ✅ 在默认浏览器中打开（右上角快捷按钮）
- ✅ 右键菜单：新建标签页、外部浏览器打开等
- ✅ 排序功能：按名称、添加时间、访问频率排序
- ✅ 搜索过滤：按文件夹名、网址名、URL检索

**数据存储：** `~/.statbox/bookmarks.json`

```json
{
  "folders": [
    {
      "id": "folder-1",
      "name": "统计学资源",
      "items": [
        {
          "id": "item-1",
          "name": "R Documentation",
          "url": "https://www.r-project.org/",
          "visitCount": 15
        }
      ]
    }
  ]
}
```

---

### 2. 💻 代码模板管理

**功能特性：**
- ✅ 支持多种语言：R / Python / Julia
- ✅ 查看模式：语法高亮显示代码
- ✅ 编辑模式：修改代码，保存前确认
- ✅ 运行功能：
  - 应用内终端执行并显示输出
  - 支持复制输出结果
  - Shiny等Web应用可在浏览器中打开
- ✅ 外部IDE打开（使用系统默认应用）
- ✅ 右键菜单：
  - 查看代码（只读）
  - 编辑代码（可修改保存）
  - 运行代码（内部终端）
  - 在外部IDE打开
- ✅ 搜索功能：按文件名、注释/描述检索

**数据存储：** `~/.statbox/templates/`

```
templates/
├── R/
│   ├── 数据清洗.R
│   └── 回归分析.R
├── Python/
│   └── pandas数据处理.py
└── Julia/
    └── 统计计算.jl
```

**模板文件示例（带注释头）：**

```r
# @name: 数据清洗模板
# @desc: 处理缺失值、异常值的标准流程
# @tags: 预处理, 数据清洗

library(dplyr)
library(tidyr)

# 代码内容...
```

---

### 3. ⌨️ 命令库

**功能特性：**
- ✅ 快捷唤醒：默认 `~` 键（可自定义）
- ✅ 智能匹配：实时搜索，匹配字符高亮显示
- ✅ 类似输入法候选框的显示方式
- ✅ 回车执行：复制命令到剪贴板
- ✅ 唤醒范围可配置：
  - 全局唤醒（在任何应用中使用）
  - 仅应用内唤醒
  - 根据文件类型智能过滤（.R, .py等）
- ✅ 应用内管理：新建、编辑、删除命令文件

**数据存储：** `~/.statbox/commands/`

```
commands/
├── LaTeX/
│   ├── 最优编译.md
│   └── 数学公式.md
├── AI提示/
│   ├── 解释统计结果.md
│   └── 优化代码.md
└── 常用/
    └── 数据分析流程.md
```

**命令文件格式（极简）：**

```markdown
description: 使用最优参数编译LaTeX文档

pdflatex -interaction=nonstopmode -shell-escape main.tex
biber main
pdflatex main.tex
```

- **文件名**：命令名称（显示在列表中）
- **description**：命令描述（显示在搜索结果中）
- **正文**：命令内容

## 🎨 界面设计

### 布局结构

```
┌─────────────────────────────────────────┐
│  [≡] StatBox  [全局搜索...]   [-] [□] [×]│
├────────┬────────────────────┬───────────┤
│ 📁收藏 │                    │ 🔍搜索... │
│ 💻代码 │   主显示面板       │ ├─统计方法│
│ ⌨️命令 │  （网页/代码编辑器/ │ ├─R代码   │
│ ⚙️设置 │   命令详情/设置）   │ └─...     │
│        │                    │           │
│[可折叠]│                    │ [可折叠]  │
└────────┴────────────────────┴───────────┘
```

### 界面特性

- ✅ **左侧面板**：模块选择器（可折叠）
  - 📁 收藏夹
  - 💻 代码模板
  - ⌨️ 命令库
  - ⚙️ 设置
  - 显示图标 + 文字
  
- ✅ **中间主面板**：显示选中项目的详细内容
  - 收藏夹模式：浏览器标签页（支持多开）
  - 代码模板模式：代码编辑器 + 终端输出
  - 命令库模式：命令详情 + 编辑界面
  - 设置模式：配置表单
  
- ✅ **右侧面板**：当前模块的列表/树（可折叠）
  - 顶部搜索框：实时过滤
  - 树形/列表结构显示
  - 可拖拽调整宽度

- ✅ **拖拽布局**：两侧面板都可折叠隐藏
- ✅ **自定义标题栏**：无边框窗口设计
- ✅ **隐藏滚动条**：保持界面简洁

## 🚀 快速开始

### 环境要求

- Go 1.21+
- Node.js 18+
- Wails CLI v2.8+

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd statbox

# 安装前端依赖
cd frontend
npm install

# 开发模式运行
wails dev

# 构建生产版本
wails build
```
