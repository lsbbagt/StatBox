# StatBox 发布版本

这个目录包含 StatBox 的编译好的可执行文件。

## 最新版本

### v1.0 (2026-03-03)

**文件：** `statbox-v1.0.exe`

**功能：**
- 📁 收藏夹管理（多标签页浏览器）
- 💻 代码模板管理（支持多层级文件夹）
- ⚙️ 系统设置（开机自启动、主题切换）
- 🎨 自定义标题栏和窗口控制

**特性：**
- 支持 exe/bat/cmd 可执行文件直接运行
- 支持 R/Python/Julia 代码文件
- 5 种颜色主题
- 自定义数据存储位置
- 静默启动功能

## 使用方法

1. 直接双击 `statbox-v1.0.exe` 运行
2. 首次运行会在 `~/.statbox/` 创建数据目录
3. 可在设置中自定义数据存储位置

## 系统要求

- Windows 10/11
- 无需安装，开箱即用
