## ✨ 核心功能模块

### 1. 📁 收藏夹管理
- 分层文件夹管理网址
- 内嵌浏览器（多标签页）
- 支持外部浏览器打开
- 排序和搜索功能
- **数据存储：** `~/.statbox/bookmarks.json`
- **状态：** ✅ 已完成

- 支持 R / Python / Julia
- 查看、编辑、运行代码
- 应用内终端输出
- 外部IDE打开

### 3. ⌨️ 命令库

- 智能匹配搜索
- 全局/局部唤醒
- 应用内管理
- **数据存储：** `~/.statbox/commands/`
- **状态：** ✅ 已完成

### 4. ⚙️ 系统设置
- 开机自启动
- 最小化到托盘
- 快捷键配置

- **状态：** ✅ 已完成

### 5. 🎨 界面特性

## 🔑 关键决策记录

### 2025-02-28
1. **界面布局选择：** 左侧模块选择器 + 中间主面板 + 右侧列表（都可折叠）
   - 理由：类似IDE布局，直观易用，支持灵活切换
   
2. **数据存储方案：** 混合模式
   - 收藏夹 → JSON（结构化数据）
   - 代码模板 → 原始文件（方便外部访问和版本控制）
   - 命令库 → Markdown（易于编辑和阅读）
   
3. **命令库格式：** 极简设计
   - 文件名作为命令名
   - 只需要 `description` 字段
   - 使用文件夹分类

4. **浏览器实现：** 默认内嵌浏览器
   - 支持多标签页
   - 提供外部浏览器打开选项
   - 可在设置中配置默认行为

5. **全局快捷键：** 使用Windows API实现
   - 支持 `~` 键默认唤醒
   - 可配置快捷键
   - 支持全局/局部唤醒范围

6. **开机自启动：** 通过Windows注册表实现
   - 使用 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
   - 支持启用/禁用/检查状态

7. **主题设计：** 温暖亮色主题
   - 主色：温暖蓝 (#5B9BD5)
   - 强调色：温暖橙 (#F4A261)
   - 背景：柔和白 (#F8FAFB)

## 💡 使用技巧

### 快速命令
- `Ctrl+B` - 显示/隐藏左侧侧边栏
- `Ctrl+Shift+B` - 显示/隐藏右侧面板
- `~` - 唤醒命令库（可自定义）

### 数据备份
建议使用 Git 管理 `~/.statbox/` 目录，便于跨设备同步。

### 开机自启动
在设置中启用"开机自启动"后，应用会自动添加到Windows启动项。
