## 🎯 项目概述

**项目名称：** StatBox - 统计学家工具箱  
**目标用户：** 统计学专业数理统计方向学生  
**核心痛点：** 代码管理分散、资源收藏混乱、重复输入频繁

## 🛠️ 技术栈

### 后端
- **语言：** Go 1.25.5
- **框架：** Wails v2.11.0
- **架构：** 清晰架构（Clean Architecture）

### 前端
- **框架：** Vue 3
- **UI库：** Vuetify 3
- **语言：** TypeScript
- **状态管理：** Pinia

### 数据存储
- **配置文件：** JSON
- **命令库：** Markdown
- **代码模板：** 原始代码文件（.R/.py/.jl）

### 系统版本（已安装）
- ✅ Go: 1.25.5
- ✅ Node.js: v22.20.0
- ✅ npm: 11.6.2
- ✅ Wails: v2.11.0

## ✨ 核心功能模块

### 1. 📁 收藏夹管理
- 分层文件夹管理网址
- 内嵌浏览器（多标签页）
- 支持外部浏览器打开
- 排序和搜索功能
- 增删改功能（右键菜单 + 按钮）
- **数据存储：** `~/.statbox/bookmarks.json`
- **状态：** ✅ 已完成

### 2. 💻 代码模板管理
- 支持 R / Python / Julia
- 查看、编辑、运行代码
- 应用内终端输出（可调整大小）
- 外部IDE打开
- **动态模块管理：** 创建、删除模块
- **文件操作：** 在文件夹中打开、删除文件
- **一键复制：** 快速复制代码
- **搜索功能：** 快速查找模板
- **数据存储：** `~/.statbox/templates/`
- **状态：** ✅ 已完成

### 3. ⚙️ 系统设置
- 开机自启动（自动静默启动）
- 最小化到托盘
- 多种颜色主题（5种可选）
- **数据存储位置配置：** 可自定义数据目录，方便备份和分享
- **状态：** ✅ 已完成

### 4. 🎨 界面特性
- 自定义标题栏（可拖动窗口）
- 无边框窗口
- 可折叠侧边栏
- 多种颜色主题
- **状态：** ✅ 已完成

## 🗂️ 项目结构

```
statbox/
├── internal/services/          # 后端服务
│   ├── config_service.go       ✅ 配置管理
│   ├── startup_service.go      ✅ 开机自启动（支持静默）
│   └── template_service.go     ✅ 模板管理
├── frontend/src/               # 前端代码
│   ├── stores/                 ✅ Pinia状态管理
│   │   ├── bookmarks.ts        # 收藏夹状态
│   │   ├── browser.ts          # 浏览器状态
│   │   ├── selection.ts        # 选择状态
│   │   └── settings.ts         # 全局设置状态
│   ├── components/             ✅ Vue组件
│   │   ├── BrowserView.vue     # 浏览器视图
│   │   ├── RightPanel.vue      # 右侧面板
│   │   ├── Sidebar.vue         # 左侧导航
│   │   └── TitleBar.vue        # 自定义标题栏
│   ├── views/                  ✅ 页面视图
│   │   ├── BookmarksView.vue   # 收藏夹（多标签页）
│   │   ├── TemplatesView.vue   # 代码模板
│   │   └── SettingsView.vue    # 设置
│   ├── plugins/vuetify.ts      # Vuetify主题配置（5种颜色）
│   ├── App.vue                 # 主应用组件
│   └── main.ts                 # 应用入口
├── main.go                     # 应用入口（支持--silent参数）
├── app.go                      # 应用逻辑
└── wails.json                  # Wails配置
```

## 💡 使用技巧

### 快速命令
- `Ctrl+B` - 显示/隐藏左侧侧边栏
- `Ctrl+Shift+B` - 显示/隐藏右侧面板
- `~` - 唤醒命令库（可自定义）

### 数据备份
建议使用 Git 管理 `~/.statbox/` 目录，便于跨设备同步。

### 静默启动
启用"开机自启动"和"静默启动"后，应用会在开机时最小化到托盘。
